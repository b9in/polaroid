# POLAROID
Vanilla JS 웹 사이트
## 프로젝트 소개
프로젝트 참여 인원 : 1명

이 프로젝트는 설명과 함께 이미지를 공유할 수 있는 소셜 미디어 웹 사이트입니다. UI를 디자인하여 ejs를 이용해 템플릿을 표시 하고 Vanilla JS로 프론트 엔드를 개발하였습니다.
또, Node.js Express를 사용하여 백엔드를 처음으로 직접 구현해 백엔드 지식을 키울 수 있었습니다.
